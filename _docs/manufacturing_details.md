# CKK 製造詳細（指示書ベース）

> 業務フロー全体の概要は [business_flow.md](./business_flow.md) §7 を参照。本ドキュメントは製造ワークフローの工程一覧・使用依存・実行依存・同期設定など実装に必要な詳細を記述する。

各指示書は **材料準備 → 加工・コーティング** の順で構成する。工程の前ステップが不要な場合は順序の入れ替えが可能。一部工程は **社内** のみ、または **社内・外注** のいずれかで実施。

| 用語 | 意味 |
|------|------|
| **使用依存** | この工程をワークフローに含めてよい条件。ワークフローに含まれるべき他工程（「存在すること」が条件）。検査対象の工程は、対応する**〇〇検査・〇〇検査承認**も使用依存に含め、ワークフローに必須とする。表内は or／and で記載。 |
| **実行依存** | この工程を**開始してよい**条件。指定した工程が**すべて完了**していること。 |
| **同期** | **あり** ＝ 他工程と組み合わせて**同時に実施・記録**できる。**なし** ＝ 単独で実施・記録する。 |

**表内の略記（使用依存・実行依存）**

| 略記 | 意味 |
|------|------|
| **素材手配** | 素材出し or 素材受渡し（いずれかがワークフローに存在／完了） |
| **素材準備済み** | 素材が（研磨・定尺）であるか、または 全長合わせ が含まれる／完了している |
| **円筒検査承認** | 円筒加工検査承認 が完了 |
| **製作完了** | 溝・外周・先端（および必要に応じホーニング）など製作工程が一通り完了した状態 |
| **A or B** | A または B のいずれかでよい |
| **A and B** | A と B の両方が必要 |

## 工程一覧（CKK 実工程）

**ルール：検査対象の工程を含む指示書では、必ず対応する「〇〇検査」ステップ（および「〇〇検査承認」）をワークフローに含め、完了させる。**
例：円筒加工を含む → 円筒加工検査・円筒加工検査承認 必須。段加工を含む → 段加工検査・段加工検査承認 必須。製作完了を含む → 製作検査・製作検査承認 必須。首逃しを含む → 首逃し検査・首逃し検査承認 必須。LD を含む → LD検査 必須。

## フロー概要（順序の目安）

```
材料準備
  素材出し / 半製品出し / 素材受渡し / 製品受渡し
    → 切断 or センタレス or 円筒加工（使用・実行依存に応じて選択）
    → 円筒加工検査 → 円筒加工検査承認
    → 全長合わせ → C面

加工・コーティング
  （研磨・定尺 or 全長合わせ）＋ 円筒加工検査承認
    → マーキング / 段加工 / 溝 / 外周 / 先端 / 首逃し（使用・実行依存に応じて、同期可能なものは並行可）
    → 段加工検査 → 段加工検査承認
    → ホーニング（先端の後）
    → 製作検査 → 製作検査承認
    → 客先向け検査１（加工後） → 客先向け検査１承認（加工後）
  製作検査承認 or 製品受渡し
    → LD / コーティング（並行または順不同のケースあり）
    → LD検査（LDの後）
    → SMAP（製作検査承認・コーティング・LDがすべて完了後、複数回あり）
    → 客先向け検査２（コーティング後） → 客先向け検査２承認（コーティング後）
    → 出荷前検査（省略可）
    → 出荷
```

## 動的順序が必要な場面

工程は厳密な順番ではなく、**実行依存（前工程の完了）** を満たせば順序を変えて実施できる。設備故障・人員都合など現場の状況で順序が変わっても、品質上問題のない工程は先に進める。

**例：段加工の設備が故障した場合**

```
通常フロー: 円筒加工検査承認 → マーキング → 段加工 → 溝 → 外周 → 先端 → 製作検査承認

当日の状況:
- 段加工の設備が故障中
- 溝・外周・先端の担当者は手が空いている
- 実行依存を確認すると、溝・外周・先端は「円筒加工検査承認が完了」が条件 → 充足

対応: 段加工の設備復旧を待ちつつ、先に溝→外周→先端を実施
記録: 実際の実施順序は テンプレート順（マーキング→段加工→溝→…）と異なるが、
      完了した工程の集合で依存関係を判定するため品質・追跡に支障なし
```

順序変更は監査ログに記録し（なぜ変えたか・誰が判断したか）、ボトルネック分析に活用する。

## 補足

- **指示書** ＝ 上記の工程を組み合わせた 1 本のワークフロー。製品・ロットごとに「材料準備 → 加工・コーティング」の流れで工程を記載する。
- **検査・承認の形式**：検査工程は「〇〇検査」、承認工程は「〇〇検査承認」で統一。承認は検査完了を実行依存とする。
- **検査の必須ルール**：検査対象となる工程（円筒加工、段加工、製作、首逃し、LD など）がワークフローに含まれる場合は、必ず対応する「〇〇検査」ステップ（および「〇〇検査承認」）をワークフローに含め、完了させる。検査ステップの省略は不可。
- 使用依存・実行依存を満たせば、表にない別順序や省略・追加は指示書ごとに調整可能。
- **送り状番号**：現状は記録していない。必要に応じて記録ルールを検討する。
- **外注時**（センタレス・コーティング等）：依頼日・入荷予定日・入荷日を管理する。
