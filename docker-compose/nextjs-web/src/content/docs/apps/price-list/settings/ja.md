# 価格表 — 設定マニュアル

価格表そのものに専用の設定画面はありません。運用ルールと関連設定をまとめます。

## 単価の決まり方

- 価格表の単価は、原則として **試算（SA05）** の原価計算から登録します。
- 原価計算のロジック（計算基準・係数・材料参照価格ポリシー）は **試算計算（SY02）** で管理します。詳しくは「試算 — 設定マニュアル」を参照してください。

## 注文種別

価格表は注文種別ごとに単価を持てます。

- **本番（PRODUCTION）** / **テスト（TEST）** / **サンプル（SAMPLE, 金額 0）** / **その他（OTHER）**。
- 同一顧客・製品でも注文種別が異なれば別エントリになります。

## 有効期間と数量段階

- **有効期間**：valid_from（必須）/ valid_until（無期限は空欄）。期間はエントリ単位で管理します。
- **数量段階**：本数範囲 → 単価。全段階でエントリの有効期間・通貨を共有します。

## 権限

- 価格表の作成・編集には `price_list` 権限が必要です。営業メンバーは自分の価格表（scope OWN）、営業補佐は閲覧のみ、営業部長は全件フルに操作できます。
