# 价格表 — 设置手册

价格表本身没有专用的设置界面。此处汇总运营规则与相关设置。

## 单价的决定方式

- 价格表的单价原则上从 **试算（SA05）** 的成本计算登记而来。
- 成本计算的逻辑（计算基准、系数、材料参考价格策略）在 **试算计算（SY02）** 中管理。详见“试算 — 设置手册”。

## 订单类型

价格表可按订单类型分别设置单价：

- **正式（PRODUCTION）** / **测试（TEST）** / **样品（SAMPLE，金额为 0）** / **其他（OTHER）**。
- 即使客户与产品相同，订单类型不同也是不同的条目。

## 有效期与数量阶梯

- **有效期**：valid_from（必填）/ valid_until（留空为无期限）。按条目管理。
- **数量阶梯**：数量区间 → 单价。所有阶梯共享条目的有效期与货币。

## 权限

- 创建/编辑价格表需要 `price_list` 权限。销售成员操作自己的价格表（scope OWN），销售辅助仅可查看，销售部长可对全部进行完整操作。
