# 価格表 — 操作マニュアル

操作コード **SA01**。顧客ごとの製品単価を管理します。見積書の単価はここから自動解決されます。

## 価格表エントリ

1 エントリ = **顧客 + 製品 + 注文種別**。以下を持ちます。

- **有効期間**（valid_from / valid_until）・通貨・状態。
- **数量段階（tier）**：本数範囲ごとの単価（例：1〜99 本 = ¥250、100 本〜 = ¥230）。

顧客・製品・注文種別は識別キーで、作成後は変更できません。

## 作成方法

- **試算から登録**：試算（SA05）を確定 → 「価格表に登録」。原価計算に基づく単価が入ります（推奨）。
- **手動作成**：一覧の「新規作成」から、顧客・製品・注文種別・有効期間・段階単価を直接入力します。

## 数量段階（tier）の編集

- 詳細画面で段階を追加・編集します。各段階は **本数の下限** と **単価**（または倍率）を持ちます。
- 段階は親エントリの有効期間・通貨を共有します。

## 一覧・検索

- 顧客・製品・注文種別・有効期間・状態で絞り込めます。
- 同じ識別キーの価格表は 1 つだけです。重複作成はエラーになります（既存を編集してください）。
