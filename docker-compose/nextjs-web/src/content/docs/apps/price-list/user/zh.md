# 价格表 — 操作手册

操作码 **SA01**。管理按客户的产品单价。报价单的单价会自动从此处解析。

## 价格表条目

一个条目 = **客户 + 产品 + 订单类型**，包含：

- **有效期**（valid_from / valid_until）、货币、状态。
- **数量阶梯（tier）**：按数量区间的单价（例：1–99 件 = ¥250，100 件以上 = ¥230）。

客户、产品、订单类型为识别键，创建后不可更改。

## 创建方式

- **从试算登记**：确定试算（SA05）后点击“登记到价格表”。会填入基于成本计算的单价（推荐）。
- **手动创建**：从列表的“新建”，直接输入客户、产品、订单类型、有效期与阶梯单价。

## 编辑数量阶梯

- 在详情页添加、编辑阶梯。每个阶梯包含 **数量下限** 与 **单价**（或倍率）。
- 阶梯共享父条目的有效期与货币。

## 列表与检索

- 可按客户、产品、订单类型、有效期、状态筛选。
- 相同识别键的价格表只能有一个，重复创建会报错（请编辑现有条目）。
