# 製品項目 — 設定マニュアル

**製品項目（SY04）** は、[製品](/docs/masters/product/user) を新規作成するときの入力項目を管理する設定アプリです。2 つの画面に分かれています。

- **項目定義** … 再利用できる入力項目のライブラリ（キー・型・選択肢など）。
- **製品種別** … 項目定義を割り当てて作るテンプレート。製品作成時に種別を選ぶと、その項目が展開されます。

> このアプリは **システム権限** が必要です。設定変更は全ユーザーの製品作成画面に影響します。

## 開き方

- ホーム（システム）→ **製品項目**、または検索ボックスで `SY04`。
- 画面右上のボタンで「項目定義 ⇄ 製品種別」を行き来できます。

## 用語

- **項目定義** … 1 つの入力欄の定義。例:「表面処理」「硬度」。
- **型（かた）** … 項目に入れられる値の種類。入力チェックに使います。
- **製品種別** … 製品テンプレート。項目定義を割り当てて構成します。例:「標準品」「コーティング品」。
- **割り当て** … 種別に項目定義を紐づけること。割り当てごとに既定値を上書きできます。

## 1. 項目定義（ライブラリ）

`/settings/product-items`（メイン画面）で、再利用する入力項目を作ります。一覧では有効/無効の切替・並び替え・削除ができ、各項目の編集は「編集」から専用ページで行います。

各項目で設定する内容:

- **項目名（日本語／英語）** … 製品作成画面に表示されるラベル。
- **キー（識別子）** … 保存時のデータ名。英字またはアンダースコアで始まる識別子（例: `surfaceTreatment`）。全体で一意。**作成後は変更できません**（種別の割り当てが参照するため）。
- **型** … 下記のいずれか。
- **必須** … オンにするとその項目を空のまま保存できません。
- **既定値（基本）** … 種別への割り当て時に上書きできます。

### 型（フォーマット）と検証

- **文字列** … 自由入力。必須なら空欄不可。
- **数値** … 数字のみ。最小値・最大値の範囲もチェックします。
- **真偽** … はい/いいえ をスイッチで選択。
- **選択** … 用意した選択肢から選ぶ。選択肢以外は不可。
- **日付** … 正しい日付であること。

## 2. 製品種別（テンプレート）

`/settings/product-items/types` で製品種別を作ります。一覧では有効/無効・並び替え・削除、編集は専用ページで行います。

種別の編集ページで設定する内容:

- **種別名（日本語／英語）／説明／有効**。
- **割り当て項目** … 「項目を割り当て」で項目定義を選び、必要なら **既定値** を上書きします（空欄なら項目定義の既定値を使います）。並び替え・削除も可能。

> 割り当てできるのは有効な項目定義だけです。項目が無い場合は先に「項目定義」で作成してください。

## 製品作成での使われ方

[製品](/docs/masters/product/user) の新規作成画面に「製品種別」欄が出ます。種別を選ぶと、割り当てられた項目が型付きで展開され、既定値が入ります。値は製品の「仕様（spec）」として保存され、選んだ種別も記録されます。種別に含まれない項目は「その他の仕様」で自由に追加できます。

> 入力が型に合わないと、保存時にエラーになり修正を促します（画面と保存処理の両方でチェックします）。

## 困ったときは

- **種別が選べない** … その種別が「無効」になっていないか確認してください。
- **割り当てる項目が無い** … 先に「項目定義」で項目を作成してください。
- **キーを変更したい** … キーは作成後に変更できません。新しい項目を作り直してください。
- 製品そのものの操作は [製品の操作マニュアル](/docs/masters/product/user) を参照してください。
