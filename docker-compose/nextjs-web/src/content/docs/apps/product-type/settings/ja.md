# 製品種別 — 設定マニュアル

**製品種別（SY04）** は、[製品](/docs/masters/product/user) を新規作成するときに使う「テンプレート」を管理する設定アプリです。種別ごとに **予め決めた入力項目** を定義しておくと、製品作成時に種別を選ぶだけでその項目が展開され、入力を型（文字列・数値など）で自動チェックできます。

> このアプリは **システム権限** が必要です。設定変更は全ユーザーの製品作成画面に影響します。

## 開き方

- ホーム（システム）→ **製品種別**、または検索ボックスで `SY04`。
- システム設定（`SY01`）の「アプリ設定」からも開けます。

## 用語

- **種別（しゅべつ）** … 製品のテンプレート。例:「標準品」「コーティング品」。
- **項目（こうもく）** … 種別が持つ入力欄。例:「表面処理」「硬度」。
- **型（かた）** … 項目に入れられる値の種類。入力チェックに使います。

## 基本操作

種別の一覧を編集し、最後に **保存** を押すと反映されます。

1. **種別を追加** … 一覧の下の「種別を追加」。種別名（日本語／英語）と説明を入力します。
2. **有効/無効** … 各種別の「有効」スイッチ。無効にすると製品作成の選択肢に出ません。
3. **並び替え** … 各行の上下ボタン。上にあるものほど先に表示されます。
4. **削除** … ゴミ箱ボタン。

## 入力項目の設定

各種別の中で「項目を追加」を押し、項目ごとに次を設定します。

- **項目名（日本語／英語）** … 製品作成画面に表示されるラベル。
- **キー（識別子）** … 保存時のデータ名。英字またはアンダースコアで始まる識別子（例: `surfaceTreatment`）。同じ種別内で重複できません。
- **型** … 次のいずれか。
- **必須** … オンにすると、その項目を空のまま保存できません。
- **既定値** … 製品作成時に最初から入っている値（任意）。

### 型（フォーマット）と検証

- **文字列** … 自由入力。必須なら空欄不可。
- **数値** … 数字のみ。最小値・最大値の範囲もチェックします。
- **真偽** … はい/いいえ をスイッチで選択。
- **選択** … 用意した選択肢から選ぶ。選択肢以外は不可。
- **日付** … 正しい日付であること。

- **数値** を選ぶと「最小値／最大値」を設定できます。
- **選択** を選ぶと「選択肢」（値と表示ラベルの組）を追加できます。値が実際に保存され、ラベルは画面表示用です。

## 製品作成での使われ方

[製品](/docs/masters/product/user) の新規作成画面に「製品種別」欄が出ます。種別を選ぶと、その種別の項目が型付きで展開され、既定値が入ります。値は製品の「仕様（spec）」として保存され、選んだ種別も記録されます。種別に含まれない項目は「その他の仕様」で自由に追加できます。

> 入力が型に合わないと、保存時にエラーになり修正を促します（画面と保存処理の両方でチェックします）。

## 困ったときは

- **種別が選べない** … その種別が「無効」になっていないか確認してください。
- **保存できない** … キーが識別子（英字/アンダースコア始まり）か、種別内で重複していないかを確認してください。
- 製品そのものの操作は [製品の操作マニュアル](/docs/masters/product/user) を参照してください。
