# 产品项目 — 设置手册

**产品项目（SY04）** 是管理新建[产品](/docs/masters/product/user)时输入项目的设置应用，分为两个界面。

- **项目定义** … 可复用的输入项目库（键、类型、选项等）。
- **产品种别** … 通过分配项目定义组成的模板。新建产品时选择种别即可展开其项目。

> 本应用需要 **系统权限**。修改会影响所有用户的产品新建界面。

## 打开方式

- 主页（系统）→ **製品項目（产品项目）**，或在搜索框输入 `SY04`。
- 右上角按钮可在"项目定义 ⇄ 产品种别"之间切换。

## 术语

- **项目定义** … 单个输入栏的定义。例如"表面处理""硬度"。
- **类型** … 项目可输入的值的种类；用于校验。
- **产品种别** … 产品模板，通过分配项目定义组成。例如"标准品""涂层品"。
- **分配** … 把项目定义关联到种别；每个分配可覆盖默认值。

## 1. 项目定义（库）

在 `/settings/product-items`（主界面）创建可复用的输入项目。列表支持启用/停用、排序、删除；各项目在专用页面通过"编辑"修改。

每个项目：

- **项目名（中/英）** … 显示在产品表单上的标签。
- **键（标识符）** … 保存时的数据名。以字母/下划线开头的标识符（如 `surfaceTreatment`），全局唯一。**创建后不可更改**（分配会引用它）。
- **类型** … 下列之一。
- **必填** … 开启后该项目不能留空。
- **默认值（基础）** … 分配到种别时可覆盖。

### 类型（格式）与校验

- **文本** … 自由输入；必填则不可为空。（可用正则表达式限制格式）
- **数值** … 仅数字；同时校验最小/最大范围。
- **布尔** … 用开关选择 是/否。
- **选择** … 从预设选项中选择；不允许其他值。
- **日期** … 须为有效日期。

## 2. 产品种别（模板）

在 `/settings/product-types` 创建产品种别。列表支持启用/停用、排序、删除；各种别在专用页面编辑。

在种别编辑页：

- **种别名（中/英）／说明／启用**。
- **分配项目** … 用"分配项目"选择项目定义，并可覆盖其 **默认值**（留空则用项目定义的默认值）。支持排序与删除。

> 只能分配已启用的项目定义。若没有项目，请先在"项目定义"中创建。

## 在产品新建中的使用

[产品](/docs/masters/product/user)新建界面会显示"产品种别"栏。选择种别后，其分配的项目会按类型展开并填入默认值。值将作为产品的规格（spec）保存，所选种别也会被记录。种别未包含的项目仍可在"其他规格"中自由添加。

> 若输入不符合类型，保存会报错并提示修正（界面与服务器都会校验）。

## 遇到问题时

- **无法选择某种别** … 确认它是否被"停用"。
- **没有可分配的项目** … 请先创建项目定义。
- **想更改键** … 键创建后不可更改；请新建一个项目。
- 产品本身的操作请参见[产品操作手册](/docs/masters/product/user)。
