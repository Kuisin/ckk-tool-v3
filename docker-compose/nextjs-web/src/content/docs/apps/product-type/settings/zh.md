# 产品种别 — 设置手册

**产品种别（SY04）** 是管理"模板"的设置应用，用于新建[产品](/docs/masters/product/user)。为每个种别预先定义一组 **输入项目**，新建产品时选择种别即可自动展开这些项目，并按格式（文本、数值等）校验输入。

> 本应用需要 **系统权限**。修改会影响所有用户的产品新建界面。

## 打开方式

- 主页（系统）→ **製品種別（产品种别）**，或在搜索框输入 `SY04`。
- 也可从系统设置（`SY01`）→ 应用设置进入。

## 术语

- **种别** … 产品模板。例如"标准品""涂层品"。
- **项目** … 种别的输入栏。例如"表面处理""硬度"。
- **类型** … 项目可输入的值的种类；用于校验。

## 基本操作

编辑种别列表，最后按 **保存** 生效。

1. **添加种别** … 列表下方的"種別を追加"。输入种别名（中/英）与说明。
2. **启用/停用** … 各种别的"有効"开关。停用后不出现在产品新建的选项中。
3. **排序** … 每行的上下按钮；越靠上越先显示。
4. **删除** … 垃圾桶按钮。

## 项目设置

在种别内按"項目を追加"，为每个项目设置：

- **项目名（中/英）** … 显示在产品表单上的标签。
- **键（标识符）** … 保存时的数据名。须为以字母或下划线开头的标识符（如 `surfaceTreatment`），且在同一种别内不可重复。
- **类型** … 下列之一。
- **必填** … 开启后该项目不能留空。
- **默认值** … 新建产品时预先填入的值（可选）。

### 类型（格式）与校验

- **文本** … 自由输入；必填则不可为空。
- **数值** … 仅数字；同时校验最小/最大范围。
- **布尔** … 用开关选择 是/否。
- **选择** … 从预设选项中选择；不允许其他值。
- **日期** … 须为有效日期。

- 选择 **数值** 可设置最小值/最大值。
- 选择 **选择** 可添加选项（值 + 显示标签）。保存的是值，标签仅用于显示。

## 在产品新建中的使用

[产品](/docs/masters/product/user)新建界面会显示"製品種別（产品种别）"栏。选择种别后，其项目会按类型展开并填入默认值。值将作为产品的规格（spec）保存，所选种别也会被记录。种别未包含的项目仍可在"その他の仕様（其他规格）"中自由添加。

> 若输入不符合类型，保存会报错并提示修正（界面与服务器都会校验）。

## 遇到问题时

- **无法选择某种别** … 确认它是否被"停用"。
- **无法保存** … 确认键是否为合法标识符（以字母/下划线开头）且在种别内不重复。
- 产品本身的操作请参见[产品操作手册](/docs/masters/product/user)。
