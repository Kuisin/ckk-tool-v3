# 見積書 — 設定マニュアル

見積書に専用の設定画面はありません。関連する設定・運用ルールをまとめます。

## 単価の自動解決

- 見積明細の単価は **価格表（SA01）** から自動で解決されます（顧客 × 製品 × 注文種別 × 数量）。
- 価格表が無い、または数量が段階に該当しない場合は単価が入りません。手動入力するか、先に価格表を整備してください。
- 価格の元となる原価ロジックは **試算計算（SY02）** で管理します。

## 採番

- 見積番号は **QOT-YYYYMM-NNNNN** 形式で、月次でリセットされます（自動採番）。

## PDF

- 見積書 PDF はシステムのテンプレートから生成されます。ロゴ・レイアウトの変更は情報システム部門へ依頼してください。
- 多言語項目（製品名など）は日本語・英語の両方が登録されている前提で表示されます。

## 権限

- 見積書の作成・編集には `quote` 権限が必要です。営業メンバーは自分の見積（scope OWN）、営業補佐は閲覧のみ、営業部長は全件フル + 承認が可能です。
