# 报价单 — 设置手册

报价单没有专用的设置界面。此处汇总相关设置与运营规则。

## 单价的自动解析

- 报价明细的单价会自动从 **价格表（SA01）** 解析（客户 × 产品 × 订单类型 × 数量）。
- 若不存在价格表，或数量不落在任何阶梯内，则单价为空。请手动输入，或先完善价格表。
- 价格背后的成本逻辑在 **试算计算（SY02）** 中管理。

## 编号

- 报价编号采用 **QOT-YYYYMM-NNNNN** 格式，按月重置（自动编号）。

## PDF

- 报价单 PDF 由系统模板生成。如需更改 Logo 或版式，请联系信息系统部门。
- 多语言字段（如产品名称）在日文与英文均已登记的前提下显示。

## 权限

- 创建/编辑报价单需要 `quote` 权限。销售成员操作自己的报价（scope OWN），销售辅助仅可查看，销售部长可对全部进行完整操作并可审批。
