# 見積書 — 操作マニュアル

操作コード **SA02**。[顧客](/docs/masters/customer/user)へ提示する見積書を作成・発行します。単価は価格表から自動解決されます。

## このアプリでできること

お客様に提出する **見積書** を作り、PDF で発行します。明細に[製品](/docs/masters/product/user)と数量を入れると、**価格表（SA01）** から単価が自動で入ります。

- 事前に価格表が登録されていれば、製品と数量を選ぶだけで金額が計算されます。
- 値引き額や納期は明細ごとに設定できます。
- 作成後は PDF を出力してお客様へ提出し、状態（下書き → 発行済 → 受諾 など）で進捗を管理します。

## 新規見積書の作成

1. 一覧の「新規作成」を押します。
2. **顧客**（必要に応じて **支店**）を選びます。
3. **明細** を追加します。行ごとに以下を入力します。
   - **製品** と **注文種別**、**数量**。
   - **単価** は顧客 × 製品 × 注文種別 × 数量から価格表の段階単価が自動でセットされます（手動上書きも可）。
   - 必要に応じて **値引き額** を入力します。金額 = 単価 × 数量 − 値引き。
   - **納期** を入力します。
4. 下部に合計金額が表示されます。

## 発行

- **保存** で下書き（DRAFT）として登録します。
- **PDF** ボタンで見積書 PDF を生成します。
- 顧客へ提示したら状態を **発行済（ISSUED）** にします。以降 **受諾（ACCEPTED）** / **却下（REJECTED）** / **期限切れ（EXPIRED）** を記録できます。

## 有効期限

- 見積書には有効期限を設定できます。期限を過ぎた見積書は EXPIRED として扱えます。

## 一覧・検索

- 見積番号（QOT-YYYYMM-NNNNN）・顧客・有効期限・状態で絞り込めます。
