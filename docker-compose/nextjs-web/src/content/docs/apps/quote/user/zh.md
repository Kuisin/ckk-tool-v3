# 报价单 — 操作手册

操作码 **SA02**。创建并发行提交给[客户](/docs/masters/customer/user)的报价单。单价会自动从价格表解析。

## 本应用能做什么

制作提交给客户的 **报价单** 并导出 PDF 发行。在明细中输入[产品](/docs/masters/product/user)与数量后，单价会从 **价格表（SA01）** 自动填入。

- 若已登记价格表，只需选择产品与数量即可计算金额。
- 折扣金额与交期可按明细分别设置。
- 创建后导出 PDF 提交给客户，并通过状态（草稿 → 已发行 → 已接受 等）管理进度。

## 新建报价单

1. 点击列表的“新建”。
2. 选择 **客户**（如需要，选择 **分店**）。
3. 添加 **明细**。每行输入：
   - **产品**、**订单类型**、**数量**。
   - **单价** 会根据 客户 × 产品 × 订单类型 × 数量 自动填入价格表的阶梯单价（可手动覆盖）。
   - 如有需要输入 **折扣金额**。金额 = 单价 × 数量 − 折扣。
   - **交期**。
4. 底部显示合计金额。

## 发行

- **保存** 会以草稿（DRAFT）登记。
- **PDF** 按钮可生成报价单 PDF。
- 提交给客户后，将状态设为 **已发行（ISSUED）**。之后可记录 **已接受（ACCEPTED）** / **已拒绝（REJECTED）** / **已过期（EXPIRED）**。

## 有效期

- 报价单可设置有效期。超过有效期的报价单可视为 EXPIRED。

## 列表与检索

- 可按报价编号（QOT-YYYYMM-NNNNN）、客户、有效期、状态筛选。
