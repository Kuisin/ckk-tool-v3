# 試算 — 設定マニュアル

操作コード **SY02**（試算計算）。試算の計算ロジックを管理者が設定します。**system 権限** が必要です。

## 計算基準（自由設定）

見積単価は **計算基準（criteria）の合計** で決まります。各基準は入力変数を使った JavaScript 式です。

- メイン画面に基準が一覧表示されます。**並び替え・有効/無効・削除・追加** はここで行います。
- 式の編集は各行の **「編集」** から専用ページで行います。
- 役割：**加算（component）** は合計に足す / **中間（intermediate）** は合計に含めず `r.<id>` で参照 / **見積単価（final）** は合計を見積単価に変換します。
- 式では入力項目・カスタム入力・`quantity`・`subtotal`・`r.<id>`・`round()` などのヘルパーが使えます。
- 「既定に戻す」で従来のロジックへ戻せます。

## 適用工具種

- 各基準に **適用工具種**（丸棒 / 円筒 / OH付）を指定できます。
- **未選択 = どの工具種にも適用されません。** 全てに適用するには全て選択してください（既定は全選択）。

## カスタム入力項目

- 試算フォームに表示する追加入力を定義できます（キー・ラベル・型・既定値）。
- キーは計算基準の式で **変数** として使えます。予約語・重複キーは保存時に弾かれます。

## 既定値・係数

- 加工単価（¥/10分）、予備形状本数、補正値、LDチャージ など、全社共通の既定値を設定します。
- 材料参照価格ポリシー（最高 / 最新 / 平均・参照月数）もここで設定します。

## カスタム計算（JS 後処理）

- 計算基準の結果に、さらに JavaScript の後処理を加えられます（見積単価の上書き・警告）。信頼できるコードのみ設定してください。
