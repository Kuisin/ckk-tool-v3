# 试算 — 设置手册

操作码 **SY02**（试算计算）。由管理员配置试算的计算逻辑，需要 **`system` 权限**。

## 计算基准（自由设置）

报价单价由 **各计算基准（criteria）之和** 决定。每个基准是使用输入变量的 JavaScript 表达式。

- 主界面以列表显示各基准。**排序、启用/停用、删除、添加** 均在此进行。
- 表达式的编辑通过每行的 **“编辑”** 在专用页面进行。
- 角色：**加算（component）** 计入合计 / **中间（intermediate）** 不计入合计但可用 `r.<id>` 引用 / **报价单价（final）** 将合计转换为单价。
- 表达式可使用输入项、自定义输入、`quantity`、`subtotal`、`r.<id>` 以及 `round()` 等辅助函数。
- 可用“恢复默认”还原为原有逻辑。

## 适用工具种类

- 每个基准可指定 **适用工具种类**（圆棒 / 圆筒 / 带OH）。
- **未选择 = 不适用于任何工具种类。** 若要适用全部，请全选（默认全选）。

## 自定义输入项目

- 可定义在试算表单中显示的额外输入（键、标签、类型、默认值）。
- 键可作为 **变量** 在计算基准表达式中使用。保留字或重复键在保存时会被拒绝。

## 默认值与系数

- 设置全公司通用的默认值：加工单价（¥/10分）、备用成型数量、修正值、LD 费用等。
- 也在此设置材料参考价格策略（最高 / 最新 / 平均、参考月数）。

## 自定义计算（JS 后处理）

- 可对计算基准的结果再施加 JavaScript 后处理（覆盖单价、添加警告）。请仅设置可信代码。
