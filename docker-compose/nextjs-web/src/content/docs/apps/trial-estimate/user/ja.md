# 試算 — 操作マニュアル

操作コード **SA05**。原価から見積単価を計算し、価格表登録の起点にするアプリです。

## このアプリでできること

「この[製品](/docs/masters/product/user)をいくらで売るか」を、材料費・加工費・コート代などの **原価** から自動計算します。出した単価は価格表に登録して、見積書で使います。

- **材料費** は素材の仕入実績（＝過去いくらで買ったか＝参照単価）から自動で入ります。
- 寸法や加工条件を入力すると、画面右側に **原価の内訳** と **見積単価** がその場で表示されます。
- はじめての方は、まず[顧客](/docs/masters/customer/user)と素材を選び、最大径・全長を入れて数字が出ることを確認してみましょう。用語は「スタートマニュアル」の用語集を参照してください。

## 新規試算の作成

1. 一覧右上の「新規作成」を押します。
2. **工具種** を選びます（丸棒 / 円筒 / OH付）。工具種により入力項目が変わります。
3. **顧客** と **素材** を選びます。素材を選ぶと仕入実績から **参照単価** が自動でセットされます。
4. 寸法・加工条件を入力します（最大径・全長・段加工・首下・コート・ラップ・LD・加工時間など）。
5. 管理者が追加した **カスタム項目** があれば入力します。
6. **基準数量** を入力します（形状出しの按分に使用）。

入力に応じて右側の **原価内訳** と **見積単価** がその場で再計算されます。

## 参照単価の上書き

- 既定では仕入実績（購買履歴）から算出した参照単価を使います。
- 手動で単価を変えたい場合は「カスタム設定」を押し、値を入力します。カスタム単価を使った試算は「カスタム」として記録されます。

## 保存・確定・価格表登録

- **保存** すると下書き（DRAFT）として登録され、その時点の価格がスナップショットとして記録されます（後で計算ロジックを変更しても、この試算の価格は変わりません）。
- 詳細画面で **確定（CONFIRMED）** にすると、価格表へ登録できるようになります。
- **価格表に登録** で、顧客・製品・注文種別を指定して価格表エントリを作成します（REGISTERED にロック）。

## 一覧・検索

- 一覧では試算番号・顧客・工具種・単価・状態で絞り込めます。
- 行をクリックすると詳細（原価内訳・ロット別単価・履歴）を確認できます。
