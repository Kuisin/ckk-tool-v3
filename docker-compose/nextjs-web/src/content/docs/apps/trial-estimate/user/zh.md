# 试算 — 操作手册

操作码 **SA05**。根据成本计算报价单价，是登记价格表的起点。

## 本应用能做什么

根据 **成本**（材料费、加工费、涂层费等）自动计算“这个[产品](/docs/masters/product/user)卖多少钱”。得出的单价登记到价格表，并在报价单中使用。

- **材料费** 会根据素材的采购实绩（即过去的采购价 = 参考单价）自动填入。
- 输入尺寸与加工条件时，画面右侧会实时显示 **成本明细** 与 **报价单价**。
- 初次使用：先选择[客户](/docs/masters/customer/user)与素材，输入最大直径与全长，确认能出数字。术语请参见“开始手册”的术语表。

## 新建试算

1. 点击列表右上角的“新建”。
2. 选择 **工具种类**（圆棒 / 圆筒 / 带OH）。不同工具种类的输入项不同。
3. 选择 **客户** 与 **素材**。选择素材后会根据采购实绩自动填入 **参考单价**。
4. 输入尺寸与加工条件（最大直径、全长、台阶加工、颈下、涂层、研磨、LD、加工时间等）。
5. 如有管理员添加的 **自定义项目**，请一并填写。
6. 输入 **基准数量**（用于分摊成型费用）。

右侧的 **成本明细** 与 **报价单价** 会随输入实时重新计算。

## 覆盖参考单价

- 默认使用根据采购实绩算出的参考单价。
- 如需手动修改，点击“自定义”并输入数值。使用自定义单价的试算会被标记为“自定义”。

## 保存、确定、登记价格表

- **保存** 会以草稿（DRAFT）登记，并将当时的价格记录为快照（即便之后更改计算逻辑，该试算的价格也不变）。
- 在详情页 **确定（CONFIRMED）** 后即可登记到价格表。
- **登记到价格表** 可指定客户、产品、订单类型创建价格表条目（锁定为 REGISTERED）。

## 列表与检索

- 可按试算编号、客户、工具种类、单价、状态筛选列表。
- 点击行可查看详情（成本明细、按批量单价、历史）。
