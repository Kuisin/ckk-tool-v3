# 承認グループ — 操作マニュアル

操作コード **MS0A**。承認を行う人のまとまり（グループ）を登録・管理します。指示書や素材発注などの承認は、このグループのメンバーが行います。

## このアプリでできること

「誰が承認できるか」をグループとして管理します。承認の段階（第一・第二など）ごとにグループを作り、メンバーを登録します。

- グループには **種別** があり、承認の役割が決まります。
- グループに **メンバー**（承認できる人）を登録します。
- 担当者が不在のときのために、期間を決めて **代理** を設定できます。

## 種別

グループの種別は 3 つです。作成時に選び、あとから変更できません。

- **第一承認** … 生産判断など、最初の承認。
- **第二承認** … 部門承認など、2 段階目の承認。
- **ワークフロー変更承認** … 製造ワークフローの変更を承認する専用グループ。

## 一覧を見る

- 一覧には登録済みのグループが並びます。行をクリックすると詳細画面が開きます。
- 種別や状態（有効 / 無効）で絞り込めます。

## 新規作成

一覧右上の「新規作成」から登録します。

- **種別** … 上記 3 つから選びます（作成後は変更不可）。
- **名称**（日本語は必須、英語は任意）。
- **有効** … オフにすると使われなくなります。

## 詳細画面

詳細画面はタブに分かれています。

- **グループ情報** … 名称・種別の確認。
- **メンバー** … 承認できる人の一覧。「メンバーを追加」で氏名・ユーザー名から検索して追加します。各メンバーは **有効 / 無効** の切り替えや **削除** ができます。
- **代理設定** … 期間限定の代理を管理します（下記参照）。
- **履歴** … 変更の記録。

右上のメニューから **編集** / **無効化** / **削除** ができます。

## 代理設定

担当の承認者が不在のとき、別の人に一時的に承認を任せる設定です。「代理設定を追加」から登録します。

- **原承認者** … 本来の承認者（グループの有効メンバーから選びます）。
- **代理人** … 代わりに承認する人。
- **開始日 / 終了日** … 代理が有効な期間。
- **理由** … 任意で記録できます。

期間を過ぎると代理は自動的に効かなくなります。

## 用語

- **承認** … 内容を確認して「進めてよい」と認めること。
- **第一承認 / 第二承認** … 承認の段階。第一が通ったら第二へ進みます。
- **メンバー** … そのグループで承認できる人。
- **代理** … 承認者が不在のとき、期間を決めて代わりに承認する人。
