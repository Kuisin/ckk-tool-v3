# 审批组 — 操作手册

操作码 **MS0A**。用于登记和管理进行审批的人员集合（组）。指示书、素材采购等的审批由该组的成员进行。

## 本应用能做什么

以组的形式管理“谁可以审批”。按审批阶段（第一、第二等）创建组并登记成员。

- 组有 **类型**，决定其审批角色。
- 向组登记 **成员**（可审批的人）。
- 为应对负责人不在，可设定限期的 **代理**。

## 类型

组的类型有 3 种。在创建时选择，之后不可更改。

- **第一审批** … 生产判断等的首次审批。
- **第二审批** … 部门审批等第二阶段的审批。
- **工作流变更审批** … 专门审批制造工作流变更的组。

## 查看列表

- 列表显示已登记的组。点击行可打开详情页。
- 可按类型与状态（有效 / 无效）筛选。

## 新建

从列表右上的“新建”登记。

- **类型** … 从上述 3 种中选择（创建后不可更改）。
- **名称**（日文必填，英文选填）。
- **有效** … 关闭后将不再被使用。

## 详情页

详情页分为多个标签。

- **组信息** … 确认名称与类型。
- **成员** … 可审批人员的列表。通过“添加成员”按姓名或用户名检索并添加。每个成员可切换 **有效 / 无效** 或 **删除**。
- **代理设置** … 管理限期代理（见下文）。
- **历史** … 变更记录。

通过右上菜单可进行 **编辑** / **停用** / **删除**。

## 代理设置

当指定的审批人不在时，可让他人临时代为审批。通过“添加代理设置”登记。

- **原审批人** … 本来的审批人（从组的有效成员中选择）。
- **代理人** … 代为审批的人。
- **开始日 / 结束日** … 代理有效的期间。
- **理由** … 可选记录。

超过期间后，代理将自动失效。

## 术语

- **审批** … 确认内容并认可“可以推进”。
- **第一审批 / 第二审批** … 审批阶段。第一通过后进入第二。
- **成员** … 在该组中可审批的人。
- **代理** … 审批人不在时，在设定期间内代为审批的人。
