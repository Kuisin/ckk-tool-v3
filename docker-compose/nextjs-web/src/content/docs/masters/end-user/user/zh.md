# 最终用户 — 操作手册

操作码 **MS02**。用于登记和管理实际使用产品的公司（最终用户）的台账。

## 本应用能做什么

与下单的[客户](/docs/masters/customer/user)（受订方）不同，当您想记录 **实际使用** 该产品的公司时使用。

- 登记为 **可选**。主要用于大额交易中需要妥善管理需求方的情形。
- 最终用户在系统内作为“业务伙伴（BP）”管理，其中带有 **需求方角色** 的即为最终用户。

## 查看列表

- 列表显示已登记的最终用户。点击行可打开详情页。
- 使用搜索框，按 **公司名称・代码** 筛选。也可按状态（有效 / 无效）筛选。

## 新建

从列表右上的“新建”登记。输入项与客户几乎相同。

**基本信息**

- **名称**（日文必填，英文选填）/ **国家** / **假名读音** / **简称** / **法人编号** / **有效**。
- **AI 匹配名** … 订单文档自动读取时用于匹配公司名的别名列表。

**地址・联系方式**

- **邮编** / **地址** / **电话** / **FAX** / **邮箱** / **网站** / **备注**。

**需求方信息**

- **行业** … 例：汽车零部件。

**BP 代码**（`BP-NNNNN`）在保存时自动编号。

## 详情页

- 显示基本信息、需求方信息、联系人等。
- 通过右上菜单可进行 **编辑** / **停用** / **删除**。

## 术语

- **最终用户** … 实际使用产品的公司，可能与下单的客户不同。
- **行业** … 该公司的经营领域。
- **BP（业务伙伴）** … 将客户、供应商、需求方等统一管理的公司单位。
