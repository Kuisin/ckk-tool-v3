# 製品 — 操作マニュアル

操作コード **MS03**。自社で作る製品を登録・管理する台帳です。[試算](/docs/apps/trial-estimate/user) や [見積書](/docs/apps/quote/user)、価格表は、ここで登録した製品を選んで作成します。

## このアプリでできること

「どんな製品を扱っているか」を登録しておきます。製品には名称のほか、どんな素材から作るか（**素材仕様**）や、寸法・公差などの **仕様** を記録できます。

- 製品を登録すると、試算・見積書・[価格表](/docs/apps/price-list/user) で選べるようになります。
- 素材は「**材種 + 直径 + 全長**」で指定します。特定の素材コードには紐付けません。

## 一覧を見る

- 一覧には登録済みの製品が並びます。行をクリックすると詳細画面が開きます。
- 検索ボックスに **製品コード・名称** を入れて絞り込めます。

## 新規作成

一覧右上の「新規作成」から登録します。主な入力項目は次のとおりです。

**基本情報**

- **名称**（日本語は必須、英語は任意）。
- **単位** … 本 / 個 / kg / m / セット から選びます。
- **有効** … オフにすると選択肢に出なくなります。
- **備考**。

**素材仕様**

- **材種** … コード構成が登録済みの材種のみ選べます。材種コード・名称で検索して選びます。
- **直径 (mm)** … 0.1〜99.9mm。入力するとコード（径 × 10）が表示されます。
- **全長 (mm)** … 1〜999mm。
- 同じ材種・直径の素材を、全長に合わせて切断して使う想定です。特定の素材には紐付けません。

**仕様**

- 「項目名」と「値」の組み合わせで自由に記述できます（例：外径 → `φ20 ±0.01`）。
- 「仕様項目を追加」で行を増やせます。

**製品コード**（`PRD-YYYYMM-NNNN`）は保存時に自動で採番されます。手入力は不要です。

## 詳細画面

- 基本情報・素材仕様・仕様（項目と値）が表示されます。
- この製品に紐付く **価格表** の一覧も確認できます。
- 右上のメニューから **編集** / **無効化** / **削除** ができます。

## 用語

- **材種** … 素材の種類（メーカー・グレード・形状などの組み合わせ）。製品は材種＋寸法で素材を指定します。
- **直径 / 全長** … 素材の太さ（mm）と長さ（mm）。
- **仕様（spec）** … 外径や公差など、製品ごとの自由記述の項目。
- **単位** … 数量の数え方（本・個 など）。

初めての方は [スタートマニュアル](/docs/start) もあわせてご覧ください。
