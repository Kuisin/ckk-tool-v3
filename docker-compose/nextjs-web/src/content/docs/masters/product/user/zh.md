# 产品 — 操作手册

操作码 **MS03**。用于登记和管理自家制作产品的台账。[试算](/docs/apps/trial-estimate/user)、[报价单](/docs/apps/quote/user) 与价格表都通过选择此处登记的产品来创建。

## 本应用能做什么

登记“经营哪些产品”。产品除名称外，还可记录由何种素材制作（**素材规格**）以及尺寸、公差等 **规格**。

- 登记产品后，即可在试算、报价单、[价格表](/docs/apps/price-list/user) 中选择。
- 素材以“**材种 + 直径 + 全长**”指定，不与特定素材代码绑定。

## 查看列表

- 列表显示已登记的产品。点击行可打开详情页。
- 使用搜索框，按 **产品代码・名称** 筛选。

## 新建

从列表右上的“新建”登记。主要输入项如下。

**基本信息**

- **名称**（日文必填，英文选填）。
- **单位** … 从 本 / 个 / kg / m / 套 中选择。
- **有效** … 关闭后不再出现在选择列表中。
- **备注**。

**素材规格**

- **材种** … 仅可选择已登记代码结构的材种。按材种代码・名称检索选择。
- **直径 (mm)** … 0.1〜99.9mm。输入后会显示代码（直径 × 10）。
- **全长 (mm)** … 1〜999mm。
- 设想是将相同材种・直径的素材按所需全长切断使用，不与特定素材绑定。

**规格**

- 以“项目名”和“值”的组合自由描述（例：外径 → `φ20 ±0.01`）。
- 通过“添加规格项目”增加行。

**产品代码**（`PRD-YYYYMM-NNNN`）在保存时自动编号，无需手动输入。

## 详情页

- 显示基本信息、素材规格、规格（项目与值）。
- 也可确认与该产品关联的 **价格表** 列表。
- 通过右上菜单可进行 **编辑** / **停用** / **删除**。

## 术语

- **材种** … 素材的种类（厂商・牌号・形状等的组合）。产品以材种＋尺寸指定素材。
- **直径 / 全长** … 素材的粗细（mm）与长度（mm）。
- **规格（spec）** … 外径、公差等各产品的自由记述项目。
- **单位** … 数量的计数方式（本・个 等）。
