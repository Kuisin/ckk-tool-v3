# Start Manual

Welcome to the CKK Business Management System. This guide helps first-time users get started.

## 1. Signing in

- Open the internal portal URL in your browser.
- Sign in with **SSO (your company account)**. Development accounts are available via the link at the bottom of the login screen.
- After your first login, check your display name and department from the avatar menu (top right).

## 2. Reading the screen

- **Logo (top left)**: opens the app launcher.
- **Center search**: jump to a screen by operation code (e.g. `SA05`) or app name.
- **Bell (top right)**: notifications. **Avatar**: profile, settings, sign out.
- **Home**: available apps grouped by category. Apps you lack permission for are hidden.

## 3. Operation codes

Every screen has a 4-character operation code. Type it in the search box and press Enter to navigate.

- `SA05` Trial Estimate / `SA01` Price List / `SA02` Quote
- `MS01` Customers / `MS02` End Users / `MS0A` Approval Groups

## 4. The sales flow

1. **Trial Estimate (SA05)** — compute a unit price from cost criteria.
2. **Price List (SA01)** — register an estimate as a customer price list.
3. **Quote (SA02)** — build and issue a quote from the price list.

See the App User Guides for details on each app.

## 5. Getting help

- Can't find a screen: you may lack the required permission — ask an administrator.
- Notifications: review them from the bell icon.
- Change your settings: use **Settings** in the top-right menu to update your notification email or password (see the User Settings Manual).
