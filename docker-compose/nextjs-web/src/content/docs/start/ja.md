# スタートマニュアル

CKK 業務管理システムへようこそ。このマニュアルは初めて利用する方向けの導入ガイドです。

## 1. ログイン

- ブラウザで社内ポータルの URL を開きます。
- **SSO（社内アカウント）** でサインインします。開発用アカウントはログイン画面下部のリンクから利用できます。
- 初回ログイン後、右上のアバターメニューから表示名・所属を確認してください。

## 2. 画面の見かた

- **ヘッダー左のロゴ**：アプリ一覧（ランチャー）が開きます。
- **中央の検索**：操作コード（例 `SA05`）やアプリ名で画面へ直接移動できます。
- **右上のベル**：通知。**アバター**：プロフィール・設定・ログアウト。
- **ホーム**：利用できるアプリがカテゴリ別に並びます。権限のないアプリは表示されません。

## 3. 操作コード

各画面には 4 文字の操作コードがあります。検索窓に入力して Enter で移動できます。

- `SA05` 試算 / `SA01` 価格表 / `SA02` 見積書
- `MS01` 顧客 / `MS02` 最終需要家 / `MS0A` 承認グループ

## 4. 販売フローの基本

1. **試算（SA05）** — 原価から見積単価を計算します。
2. **価格表（SA01）** — 試算を顧客別の価格表として登録します。
3. **見積書（SA02）** — 価格表から見積書を作成し、顧客へ発行します。

各アプリの詳しい操作は「アプリ操作マニュアル」を参照してください。

## 5. 困ったときは

- 画面が見当たらない：権限が不足している可能性があります。管理者にご相談ください。
- 通知が届く：ベルアイコンから履歴を確認できます。
- 設定を変えたい：右上メニューの「設定」から、通知先メールやパスワードを変更できます（「ユーザー設定マニュアル」参照）。
