# スタートマニュアル

CKK 業務管理システムへようこそ。このページは **初めて使う方** 向けの入門ガイドです。専門用語は最後の「用語集」で説明します。

## このシステムとは

見積・受注・製造・出荷・請求までを 1 つにまとめた社内システムです。まずは営業に関わる 3 つのアプリ（**試算・価格表・見積書**）から使い始めます。

- **試算** … 「この製品はいくらで作れるか」を原価から計算します。
- **価格表** … 顧客ごとの販売単価を登録しておく台帳です。
- **見積書** … お客様に提出する見積書を作ります。単価は価格表から自動で入ります。

この 3 つは **試算 → 価格表 → 見積書** の順につながっています（下の「4. 販売の流れ」参照）。

## 1. ログイン

1. ブラウザで社内ポータルの URL を開きます。
2. **SSO（社内アカウント）** のボタンでサインインします。開発用アカウントはログイン画面下部のリンクから使えます。
3. ログインできたら、右上の丸いアイコン（アバター）をクリックして、自分の名前・所属が正しいか確認します。

## 2. 画面の見かた

- **左上のロゴ** をクリック → 使えるアプリの一覧（ランチャー）が開きます。
- **中央の検索ボックス** … アプリ名や「操作コード」（例 `SA05`）を入力して Enter すると、その画面へ直接移動します。
- **右上のベル** … お知らせ（通知）。**アバター** … プロフィール・設定・ログアウト。
- **ホーム画面** … 使えるアプリがカテゴリ別に並びます。権限のないアプリは表示されません。

## 3. 操作コード（画面番号）

各画面には 4 文字の番号があります。覚えておくと検索ボックスからすぐ移動できます。

- `SA05` 試算 / `SA01` 価格表 / `SA02` 見積書
- `MS01` 顧客 / `MS02` 最終需要家 / `MS0A` 承認グループ

## 4. 販売の流れ（まずはこれだけ）

1. **試算（SA05）で単価を計算する** … 製品・素材・寸法などを入力すると、原価から見積単価が出ます。（[試算の操作マニュアル](/docs/apps/trial-estimate/user)）
2. **確定して価格表（SA01）へ登録する** … 試算を「確定」し、顧客・製品を指定して価格表に登録します。（[価格表の操作マニュアル](/docs/apps/price-list/user)）
3. **見積書（SA02）を作る** … 顧客と製品を選ぶと、価格表から単価が自動で入ります。PDF を出してお客様へ提出します。（[見積書の操作マニュアル](/docs/apps/quote/user)）

各アプリの詳しい操作は左の「アプリ操作マニュアル」を開いてください。

## 5. 困ったときは

- **画面が見当たらない** … 権限が足りない可能性があります。管理者に相談してください。
- **通知が来た** … 右上のベルから内容を確認できます。
- **自分の設定を変えたい** … 右上メニューの「設定」から通知先メールやパスワードを変更できます（「ユーザー設定マニュアル」参照）。

## 用語集

- **試算（しさん）** … 原価から販売単価を計算すること。見積の下準備。
- **参照単価 / 仕入実績** … 素材をいくらで仕入れたか（過去の購買履歴）。試算の材料費はこの実績価格を基に自動計算されます。
- **工具種（こうぐしゅ）** … 製品の種類。丸棒・円筒・OH付 の 3 種類があり、種類ごとに入力項目や計算式が変わります。
- **ロット** … まとめて作る本数（数量）のまとまり。数量が多いほど 1 本あたりの単価は下がります。
- **注文種別** … 本番・テスト・サンプル（金額 0）・その他 の区分。同じ製品でも種別ごとに価格を分けられます。
- **段取り / 形状出し** … 加工前の準備費用。数量で割って 1 本あたりに按分します。
- **確定 / 下書き** … 下書き（編集中）→ 確定（内容を固定）の状態。確定すると価格表に登録できます。
