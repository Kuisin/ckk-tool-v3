# 开始手册

欢迎使用 CKK 业务管理系统。本手册面向首次使用的用户。

## 1. 登录

- 在浏览器中打开公司内部门户网址。
- 使用 **SSO（公司账号）** 登录。开发用账号可通过登录页底部的链接使用。
- 首次登录后，请从右上角头像菜单确认显示名称与所属部门。

## 2. 界面说明

- **左上角 Logo**：打开应用启动器（应用列表）。
- **中间搜索框**：通过操作码（如 `SA05`）或应用名称直接跳转。
- **右上角铃铛**：通知。**头像**：个人资料、设置、退出登录。
- **主页**：按类别显示可用应用。无权限的应用不会显示。

## 3. 操作码

每个界面都有 4 位操作码。在搜索框输入并按回车即可跳转。

- `SA05` 试算 / `SA01` 价格表 / `SA02` 报价单
- `MS01` 客户 / `MS02` 最终需求方 / `MS0A` 审批组

## 4. 销售流程基础

1. **试算（SA05）** — 根据成本计算报价单价。
2. **价格表（SA01）** — 将试算登记为按客户的价格表。
3. **报价单（SA02）** — 根据价格表生成报价单并发给客户。

各应用的详细操作请参阅“应用操作手册”。

## 5. 遇到问题时

- 找不到某个界面：可能是权限不足，请联系管理员。
- 通知：可从铃铛图标查看历史记录。
- 修改设置：从右上角菜单的“设置”可更改通知邮箱与密码（参见“用户设置手册”）。
