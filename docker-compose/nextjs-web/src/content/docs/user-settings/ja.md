# ユーザー設定マニュアル

自分のアカウントに関する設定です。右上のアバターメニュー →「プロフィール」/「設定」から開きます。

## プロフィール

- **表示名・所属**：社内ディレクトリ（AD）と同期されます。変更は情報システム部門へ依頼してください。
- **メールアドレス（通知宛先）**：通知を受け取るメールアドレスを設定します。空欄の場合、メール通知は送信されません。

## パスワード

- ログインに SSO を使う場合、パスワードは社内アカウント側で管理します。
- システム内アカウントのパスワードを変更する場合は「設定 → パスワード」から行います。現在のパスワードと新しいパスワード（2 回）を入力します。

## 通知

- **アプリ内通知**：ヘッダーのベルに届きます。「すべて既読」でまとめて既読にできます。
- **メール通知**：プロフィールのメールアドレス宛に送られます。
- **プッシュ通知（PWA）**：ホーム画面に追加したアプリで有効化できます（iOS 16.4 以降）。

## 登録デバイス

- プッシュ通知を有効にしたデバイスの一覧を確認・削除できます。使わなくなった端末は削除してください。

## 言語

- マニュアル（/docs）は右上の言語リンクで日本語 / English / 中文 を切り替えられます。
