# 用户设置手册

与自己账号相关的设置。从右上角头像菜单 →“个人资料”/“设置”打开。

## 个人资料

- **显示名称 / 所属部门**：与公司目录（AD）同步。如需更改请联系信息系统部门。
- **邮箱（通知地址）**：接收通知的邮箱地址。若为空，则不发送邮件通知。

## 密码

- 若使用 SSO 登录，密码由公司账号一侧管理。
- 如需更改系统内账号的密码，请在“设置 → 密码”中操作：输入当前密码与新密码（两次）。

## 通知

- **应用内通知**：显示在页头的铃铛处，可用“全部标记为已读”统一处理。
- **邮件通知**：发送至个人资料中的邮箱。
- **推送通知（PWA）**：在添加到主屏幕的应用中启用（iOS 16.4 及以上）。

## 已注册设备

- 可查看并删除已启用推送通知的设备。请删除不再使用的终端。

## 语言

- 手册（/docs）可通过右上角的语言链接在 日本語 / English / 中文 之间切换。
